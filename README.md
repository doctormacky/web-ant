中文说明
============================

这是一个脚本项目，该脚本项目能帮助java web开发人员快速的构建一个可部署的war包

项目目录例子如下：你只需要把本build.xml拷贝到对应的目录即可

 Project
 └── Java Resources
	 ├── src
	 ├── config
	 └── Libraries
     WebContent
     ├── <web-file-folders>
     ├── WEB-INF
     |   ├── lib
     |   |   └── xxx.jar
     |   └── web.xml
     ├── build.xml
     └──README.md
